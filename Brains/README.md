# Victor Brains Library

This folder is Victor's constitutional financial-literacy layer.

The source order matters:

1. The project-wide master file remains [victor_sterling_financial_project_continuation_brief.md](/C:/Dev/Victor/victor_sterling_financial_project_continuation_brief.md).
2. This `Brains/` folder is the next layer of authority.
3. All supporting prompts, operating docs, audits, workflows, and app behavior must derive from these sources rather than override them.

## Ranked Source Hierarchy

1. *The White Coat Investor* by James M. Dahle, MD
   Role: primary physician-finance constitution
2. *The White Coat Investor's Financial Boot Camp* by James M. Dahle, MD
   Role: implementation manual
3. *The Doctors Guide to Eliminating Debt* by Cory S. Fawcett, MD
   Role: debt-discipline counterweight
4. *The Bogleheads' Guide to Investing*
   Role: broad investing doctrine
5. *The Little Book of Common Sense Investing* by John C. Bogle
   Role: index-investing constitution

## What These Sources Are For

- physician-specific financial decision-making
- debt discipline
- long-term low-cost investing
- insurance and risk management
- tax-aware wealth accumulation
- skepticism toward financial products sold to physicians

## What These Sources Are Not For

- current tax brackets
- current IRS contribution limits
- current student-loan repayment program details
- state-specific legal rules

Those items require live verification from official current sources before action.
