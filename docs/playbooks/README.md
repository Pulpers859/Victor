# Victor Workbook Playbooks

These playbooks explain how Victor should interpret the spreadsheet workflow.

They do not outrank the master brief or the constitutional sources. Their job is narrower:

- explain how to read workbook tabs
- keep spreadsheet interpretation consistent
- prevent formula outputs from being mistaken for final advice
- identify where more data or current-law verification is still required
